# Language56 Translator Bot 🤖

A powerful Telegram translation bot supporting 25+ languages, built with Python and deployed on Railway.

## 🌟 Features

- 🌍 Translate between 25+ languages
- 🚀 Auto-detects source language
- ⚡ Quick translation with "text to lang" format
- 🔄 Reply translation support
- 🎯 Default language preferences
- 📱 Interactive buttons for easy use
- 🔍 Language detection

## 🚀 Deployment on Railway

### 1. Create Telegram Bot
1. Search for `@BotFather` on Telegram
2. Send `/newbot` and follow instructions
3. Copy the API token

### 2. Deploy on Railway
1. Push this code to GitHub
2. Go to [Railway](https://railway.com)
3. Click "New Project" → "Deploy from GitHub"
4. Select your repository
5. Add Environment Variable: `TOKEN` = Your bot token

### 3. Bot Commands
- `/start` - Welcome message
- `/help` - Help guide
- `/languages` - List all languages
- `/setlang` - Set default language
- `/translate` - Translate replied message
- `/detect` - Detect language
- `/clear` - Clear preferences
